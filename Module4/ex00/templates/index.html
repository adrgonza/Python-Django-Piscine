Ex00: Markdown Cheatsheet

Headers

# H1
## H2
### H3
#### H4
##### H5
###### H6

Emphasis

*italic* or _italic_
**bold** or __bold__
~~strikethrough~~

Lists

Unordered:
- Item 1
- Item 2
- Item 3
Ordered:
1. Item 1
2. Item 2
3. Item 3

Links

[link text](url)

Images

![alt text](image url)

Code

Inline `code`
Block code:
``` code block ```

Blockquotes

> blockquote

Horizontal Rule

---
***
___

Tables

| Header 1 | Header 2 |
| --- | --- |
| Cell 1 | Cell 2 |
| Cell 3 | Cell 4 |
