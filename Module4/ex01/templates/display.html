Ex01: Display process of a static page

When a user requests a static page, Django follows these steps:

  1. The URL dispatcher maps the URL to the appropriate view.
  2. The view processes the request, accessing necessary data.
  3. The view passes data to a template, rendering the final HTML.
  4. Django returns the rendered HTML as an HTTP response.
